## 추가 실측

### 신청기간 상태 분포 (기준일 2026-07-15, Asia/Seoul)

**기업마당 1,527건** (API가 신청 가능 공고만 제공 — CLOSED 0%):
- OPEN 692 (45.3%) / ROLLING 686 (44.9%, "상시 접수"·"예산 소진시까지") / UPCOMING 18 (1.2%) / CLOSED 0 / UNKNOWN 131 (8.6%)
- UNKNOWN 원문 유형: "선착순 접수", "차수별 상이", "정원 마감시까지", "모집 완료시", "세부사업별 상이", "수시 모집" 등 → Phase 1에서 ROLLING 패턴 확장 검토 대상

**K-Startup**: 모집중 필터(rcrt_prgs_yn=Y) 261건은 전건 OPEN (100%). 전체 이력 29,430건 중 모집중 261건 → **마감·비진행 99.1%** (전체를 받으면 안 되고 반드시 모집중 필터 사용).

### 지역 필드 고유값

- **K-Startup `supt_regin`** (17개): 강원, 경기, 경남, 경북, 광주, 대구, 대전, 부산, 서울, 울산, 인천, 전국, **전남광주**(비표준 결합값), 전북, 제주, 충남, 충북 — 원본 보존 + 정규화 필요
- **기업마당**: 별도 지역 필드 없음. `hashtags` 안에 지역 토큰 포함 (전국 = 17개 시도 전체 나열 형태 316건). 지역태그 없는 공고 27건(1.8%) → region_normalized=UNKNOWN 처리
- 기업마당 해시태그 지역값도 '전남광주' 형태 확인됨 (API 문서 URL코드표 기준)

### 소스 간 중복 (정규화 제목 + 신청기간 + 기관 기준, 제목 단순 비교 아님)

- K-Startup 모집중 261건 중 기업마당과 정규화 제목 완전일치 16건(6.1%), 부분 포함 모호 후보 3건(1.1%)
- **기업마당에 없는 K-Startup 고유 공고: 242건 (92.7%)** → 채택 시 고유 증가분 큼
- 기업마당 내 소상공인시장진흥공단 소관/수행 공고 13건, trgetNm=소상공인 189건 → 소상공인마당 제외해도 소상공인 공고 커버 가능

### 결정론적 판정 가능 비율 (Phase 0 추정)

- **K-Startup**: 지역(supt_regin)·업력(biz_enyy)·연령(biz_trgt_age)·대상유형(aply_trgt) 4개 조건 모두 구조화 필드로 100% 채움 → 이 4개 조건은 100% 코드 판정 가능. 서술형 aply_trgt_ctnt(100% 존재)의 패턴 추출률은 Phase 1에서 실측
- **기업마당**: 대상유형(trgetNm, 9개 고유값) 100%, 지역(hashtags) 98.2% 판정 가능. 업력·매출·인증 등 세부 필수요건은 구조화 필드 없음 → bsnsSumryCn(요약)·첨부에서 추출 실패 시 eligibility_complete=false → ⚪ 판정보류 예상 비율 높음 (정직하게 보고할 것)

### 첨부문서 확인 필요 비율

- 기업마당: 첨부 보유 1,129/1,527 (74%), HWP·HWPX 포함 886건 (58%) → 세부 자격요건이 첨부에 있는 사업 다수
- K-Startup: 첨부 필드 대신 상세페이지 URL 100% + 서술형 자격요건 필드 제공 → 첨부 의존도 낮음

## 권고

1. **채택**: 기업마당(주 소스, 전 부처·지자체 커버) + K-Startup(창업 분야 보강, 고유 92.7%)
2. **제외**: 소상공인마당 — 공식 공고 API 없음, SPA 내부 엔드포인트 사용은 원칙 3 위반, 기업마당이 소상공인 공고 189건 커버
3. **BLOCKED → v1 제외**: 중소벤처24 — 공고정보 API는 수동 승인 필요 + 중기부 유관기관 공고는 기업마당과 중복 가능성 높음(미실측). 증명(확인)서 API는 기업회원 전용으로 공고 수집이 아닌 **프로필 보강(벤처·이노비즈·메인비즈 확인서)** 용도 → v2 후보
4. 수집 시 준수: 같은 호스트 1초 지연, 명시적 User-Agent, fetched==totCnt 검증 (기업마당 1527==1527 확인, K-Startup 261==matchCount 확인)
